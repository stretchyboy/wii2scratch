Instructions

You will need to have :-
	plugged in your bluetooth dongle
	started your bluetooth stack / gui
	discovered the wiimote
	started the HDI (Human Interface Device) service of the WiiMote
	
	Started Scratch
	possibly load wiitest.sb that will be in the installion directory.
	Gone to the sensors tab
	right click the "sensor value" block and click "enable remote sensor conections"
	
	
Then run Wii2Scratch

If its all worked your WiiMote will rumble.
if not try running Wii2Scratch.jar on the command line by going to the install directory in a dos prompt and running 
	java -jar Wii2Scratch.jar
	
this will give you more info.
